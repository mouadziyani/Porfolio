# Assets Folder

This folder contains images and other assets for the portfolio website.

## Required Images:

- `profile.jpg` - Professional headshot (400x400px recommended)
- `project1.jpg` - E-commerce Platform screenshot (800x600px recommended)
- `project2.jpg` - Task Management App screenshot (800x600px recommended)
- `project3.jpg` - Weather Dashboard screenshot (800x600px recommended)
- `favicon.ico` - Website favicon (16x16px or 32x32px)

## Image Guidelines:

- Use high-quality, professional images
- Optimize for web (compress images)
- Maintain consistent aspect ratios
- Use alt text for accessibility
- Ensure mobile responsiveness

## Placeholder Images:

You can use placeholder images from services like:
- https://picsum.photos/
- https://placeholder.com/
- https://unsplash.com/ (for production)

## File Naming:

- Use lowercase letters
- Use hyphens instead of spaces
- Be descriptive but concise
- Include dimensions if helpful (e.g., `profile-400x400.jpg`)
